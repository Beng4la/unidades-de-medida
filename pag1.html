Guía de Unidades de Medida

Medidas Absolutas y Relativas

Para las fuentes, las unidades de medida relativas como "em" y "rem" son las
mejores opciones. Esto se debe a que se ajustan automáticamente en función
del tamaño de fuente del elemento padre, lo que facilita la creación de diseños
responsivos y escalables.


Para los contenedores, las unidades de medida relativas como porcentaje (%)
son ideales para crear diseños fluidos y adaptables a diferentes tamaños
de pantalla. Esto permite que los elementos se escalen proporcionalmente
en relación con el tamaño del contenedor padre, lo que garantiza una
presentación consistente en dispositivos de diferentes tamaños.
